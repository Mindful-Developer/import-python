.. function:: count(start=0, step=1)

   Return an infinite iterator of integers starting at ``start`` (or
   0 if not provided), incremented by ``step`` (or 1 if not
   provided).  ``step`` defaults to 1.

   :param start: The starting value (default 0).
   :param step: The increment step (default 1).
   :returns: An iterator of integers.
